Enter your username, and choose the action you wish to take.

"Create" will create a user with the entered username, as long as you still have enough user slots open.

"Delete" will delete all datas for the user with this name on this navigator. It won't do anything if no user with the name entered exitsts.

"Export" will give you the saved datas of the username you entered, allowing you to import it on another navigator.

"Import" allows you to enter the saved datas exported from another navigator to be able to play on any username anywhere.
