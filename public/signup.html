Create User


"Create" will create a user with the entered username, as long as you still have enough user slots open.

Import User



"Import" allows you to enter the saved data exported from another navigator to be able to play on any username anywhere.

Current User



"Delete" will delete all data for the user with this name on this navigator. It won't do anything if no user with the name entered exitsts.

"Export" will give you the saved data of the username you entered, allowing you to import it on another navigator.
